Ogni query e ogni porzione di codice sql deve essere inserita in un metodo il cui nome, per praticità, riporta le sue coordinate ed è così composto:
nome_classe + _ + nome_metodo

Esempio

include_once(LIB_DIR.OS."sqlcode.php");
$obj = new sqlcode();
$call = get_class().'_methodName';
$query = $obj->$call($param1, $param2, $param3);
